### Requirements
- Python>=3.12
- pillow==12.1.1
- qrcode==8.2

### Description
Just a simple QR Maker. Usage is pretty simple.

```
python src/main.py --txt "Whatever you want to say"
```

This will create a file named `qr_code.png` that is readable with QReader, Phones, QR apps, etc. which can be found in the same directory.
