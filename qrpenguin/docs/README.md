# QR Penguin

Simple QR maker in Python

I named it `QR Penguin` because Penguins eat fish.

Usage is pretty simple.
```
~$: python src/main.py --txt "Whatever you want to say"
```
Will create a file named `qr_code.png` that is readable with QReader, Phones, QR apps, etc.
